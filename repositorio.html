    Bases de dados

     Novas perspectivas sobre crime e segurança pública

    Introdução

    O projeto tem como objetivo principal criar um repositório online que disponibilize o acesso a dados e documentação das pesquisas realizadas pelo CRISP ao longo dos últimos 25 anos. Além disso, o repositório será expandido para incluir outras bases de dados de acesso livre, provenientes de repositórios de gestores públicos, pesquisadores autônomos e instituições de pesquisa relacionadas à criminalidade, segurança pública e justiça criminal. Através dessa iniciativa, pretendemos promover a transparência, compartilhamento e utilização dessas informações para fins acadêmicos, científicos e de formulação de políticas públicas.

    Objetivos

    • Criar uma página da internet para o repositório de bases de dados do CRISP, com uma interface amigável e de fácil navegação.
    • Organizar, sistematizar e disponibilizar as bases de dados resultantes das pesquisas realizadas pelo CRISP nos últimos 25 anos.
    • Descrever e documentar as pesquisas, incluindo relatórios, artigos, metodologias e outros materiais relevantes, para acompanhamento e compreensão adequada dos dados.
    • Pesquisar e agregar outras bases de dados de acesso livre, relacionadas à criminalidade, segurança pública e justiça criminal, provenientes de repositórios de gestores públicos e instituições de pesquisa.
    • Elaborar termos de uso e políticas de acesso, inclusão de dados, armazenamento e segurança.
    • Garantir a segurança e confidencialidade dos dados sensíveis, adotando medidas adequadas de proteção da privacidade.
    • Desenvolver manuais de orientação para os serviços oferecidos.

    Etapas do projeto

    Planejamento e infraestrutura

    • Estabelecer rotina de reuniões com a equipe responsável pelo projeto, incluindo coordenação, desenvolvedor web, estagiário e pesquisadores envolvidos.
    • Definir a estrutura do repositório, considerando categorias, metadados e recursos de busca.
    • Estabelecer requisitos técnicos para o servidor, armazenamento e segurança dos dados.

    Organização e sistematização dos dados

    • Identificar todas as pesquisas realizadas pelo CRISP nos últimos 25 anos.
    • Elaborar uma descrição sobre as pesquisas identificadas, incluindo pesquisadores responsáveis e todos os envolvidos nas pesquisas.
    • Coletar as bases de dados relacionadas a cada pesquisa.
    • Padronizar e organizar os dados de acordo com as melhores práticas de gestão de dados.
    • Desenvolver um sistema de indexação e metadados para facilitar a busca e recuperação dos dados.

    Documentação das pesquisas

    • Reunir todos os relatórios, artigos e outros materiais produzidos referente às pesquisas do CRISP.
    • Digitalizar e disponibilizar os documentos em formato eletrônico, quando necessário e respeitando as diretrizes da Lei Geral de Proteção de Dados.
    • Criar uma seção de documentação para cada pesquisa, fornecendo informações detalhadas sobre a metodologia, amostra, variáveis utilizadas, entre outros.

    Pesquisa e inclusão de outras bases de dados

    • Identificar repositórios públicos e instituições de pesquisa que disponibilizam bases de dados relevantes para o tema.
    • Estabelecer parcerias e acordos de compartilhamento de dados, quando necessário.
    • Coletar, organizar e integrar as novas bases de dados ao repositório do CRISP.
    • Desenvolver um guia para inclusão de dados e documentação de pesquisas conduzidas por pesquisadores autônomos.

    Desenvolvimento da página web

    • Apoiar a criação de uma interface de usuário intuitiva e responsiva para facilitar a navegação e busca de dados.
    • Identificar e sugerir possibilidades de desenvolver recursos de pesquisa avançada, filtros e visualizações interativas.
    • Auxiliar no desenvolvimento das garantias de acessibilidade do site, tornando-o compatível com diferentes dispositivos e navegadores.

    Implementação dos termos de uso e políticas de segurança

    • Desenvolver políticas de privacidade, de acesso, inclusão de dados armazenamento e segurança.
    • Apoiar na implementação de medidas de segurança robustas para proteger os dados sensíveis e pessoais.
    • Utilizar criptografia para garantir a transmissão segura dos dados.
    • Estabelecer os termos de uso para os usuários do repositório.

    Testes e avaliação

    • Realizar testes de usabilidade e funcionalidade da página web.
    • Coletar feedback dos usuários e apoiar os ajustes conforme necessário.
    • Realizar uma avaliação contínua do repositório para garantir a qualidade e atualização dos dados.

    Lançamento e divulgação

    • Disponibilizar o repositório online ao público em geral.
    • Realizar campanhas de divulgação para aumentar a visibilidade e uso do repositório.
    • Promover parcerias com instituições de pesquisa, universidades e órgãos governamentais para ampliar o alcance do projeto.
